# Conrad Linus Muhirwe - Personal Portfolio

A personal portfolio website showcasing the professional journey, skills, and projects of Conrad Linus Muhirwe, a Data Scientist, Data Analyst, and AI Engineer.

## Overview

This repository contains the source code for my personal portfolio website. The site is built with HTML, CSS, and vanilla JavaScript and highlights my experience at the intersection of advanced machine learning and financial operations.

## Features

- **Responsive Design**: Fast and mobile-friendly layout optimized for all screen sizes.
- **Interactive UI**: Tab-based navigation for accessing **About Me**, **Projects**, and **Certifications** sections.
- **Modern Aesthetics**: Built with custom CSS featuring a clean, professional design block and typography.

- **FontAwesome Icons**
